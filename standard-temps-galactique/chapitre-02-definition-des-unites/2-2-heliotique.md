# 2.2 — Héliotique (HÉL)

**Définition** : `1 HÉL = 10⁻⁶ GY = 230 ans terrestres`. L’héliotique représente l’échelle temporelle pertinente pour les politiques séculaires, les programmes d’archivage et les évolutions climatiques profondes.

## Propriétés

- Durée proche d’une longue vie humaine × 3, ce qui facilite la narration intergénérationnelle.
- Suffisamment courte pour suivre les transitions énergétiques ou sociopolitiques.
- Peut être subdivisée en 1000 COS, permettant un suivi opérationnel (chapitre 2.3).

## Notation

```
HÉL 008.742
↑ ↑   ↑
| |   └─ milli-héliotiques (mHÉL)
| └──── héliotiques
└────── préfixe "HÉL"
```

La norme SUCL recommande de publier les dates longues sous la forme : `EZG + HÉL 008.742`, associée à la version TG utilisée. Les héliotiques servent de repères dans les Géodes et dans les projets de stockage nucléaire (référence Annexe A5).
