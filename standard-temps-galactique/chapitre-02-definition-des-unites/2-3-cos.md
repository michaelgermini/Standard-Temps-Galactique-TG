# 2.3 — Cycle d’Orbite Solaire (COS)

**Définition** : `1 COS = 10⁻⁹ GY ≈ 84,0 jours terrestres`. Cette unité correspond à la division la plus naturelle de l’héliotique pour la planification opérationnelle des missions scientifiques, diplomatiques ou éducatives.

## Usages principaux

1. **Calendrier civil galactique** : une année terrestre contient ~4,345 COS. Les gouvernements peuvent regrouper leurs saisons administratives autour de blocs entiers de COS pour harmoniser les échanges interplanétaires.
2. **Gestion de projets** : un COS suffit pour mener une campagne d’observation, une rotation d’équipage ou une phase de construction modulaire.
3. **Communication SUCL** : les messages codés pour les générations futures indiquent les jalons en COS afin que les destinataires puissent repérer des rythmes de court terme.

## Notation abrégée

```
COS 512.34
```

Où `512` identifie le cycle à l’intérieur d’un héliotique et `.34` représente les galaces (section 2.4). Cette écriture hiérarchique simplifie la conversion informatique (voir chapitre 6). Le standard autorise également l’écriture `0.00051234 HÉL`, utile lorsque l’on souhaite rester dans une seule unité.
