# 2.1 — Année galactique (GY)

**Définition** : 1 GY correspond à la période orbitale moyenne du Soleil autour du centre galactique, soit `2πR₀ / Θ₀`. La valeur adoptée pour TG.v2025 est `230 000 000 ans terrestres`.

## Formulation mathématique

```
GY = 2πR₀ / Θ₀
R₀ = 8,178 kpc
Θ₀ = 229 km·s⁻¹
```

La conversion vers les unités SI passe par les facteurs : `1 kpc = 3,085677581×10¹⁹ m` et `1 an terrestre = 31 556 926 s`.

## Recommandations d’usage

- Toujours noter l’incertitude associée (`±20 Myr`) dans les métadonnées.
- Mentionner la version (`TG.vYYYY`) lors d’une publication scientifique.
- Pour les récits historiques, exprimer les ères majeures en fractions de GY (ex. : `Ère Anthropocène = EZG + 0,000000013 GY`).

Le GY sert d’étalon ultime : toutes les unités dérivées en descendent, ce qui garantit la cohérence entre données astrophysiques, documents culturels et géostratégie.
