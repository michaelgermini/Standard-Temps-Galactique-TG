# Annexe A1 — Glossaire

| Terme | Définition | Remarques |
|-------|------------|-----------|
| **TG** | Temps Galactique | Référentiel décrit dans ce traité. |
| **GY** | Année galactique | 230 Myr (±20 Myr). |
| **HÉL** | Héliotique | 10⁻⁶ GY, 230 ans. |
| **COS** | Cycle d’Orbite Solaire | 10⁻⁹ GY, 84 jours. |
| **G / sG / µG** | Galace, sous-galace, microgalace | 20 h, 12 min, 7,26 s. |
| **SUCL** | Standard Universal Chrono-Lattice | Langage temporel décimal basé sur une spirale logarithmique. |
| **MULS** | Macro Universal Layering System | Taxonomie de couches d’archives. |
| **Géode GEO‑01** | Artefact d’archivage | Sphère multi-couches contenant le standard TG. |
| **EZG** | Époque Zéro Galactique | Instant où le Soleil a croisé l’axe radial vers le centre. |
| **TG.vYYYY** | Version du standard | Inclut les valeurs actualisées de GY, R₀, Θ₀. |

Ce glossaire doit être reproduit dans chaque support TG (Géodes, dépôts numériques) pour aider les lecteurs à naviguer rapidement dans la terminologie.
